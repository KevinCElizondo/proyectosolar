# Capturas de Pantalla de Solar Fluidity

Este directorio contiene capturas de pantalla organizadas de la plataforma Solar Fluidity, mostrando las diferentes secciones y funcionalidades disponibles. Las imágenes están organizadas por versión y módulo funcional.

## Estructura de Carpetas

- **hero_section/**: Capturas de la sección principal (hero) y landing page
- **facturacion/**: Sistema de facturación electrónica offline
- **proyectos/**: Gestión de proyectos solares y electromecánicos
- **automatizaciones/**: Flujos de automatización con Agentes IA / Python
- **integraciones/**: Integraciones con servicios externos
- **reportes/**: Dashboards y reportes financieros
- **ui_components/**: Componentes de interfaz de usuario

## Cómo Usar estas Capturas

Estas capturas de pantalla pueden ser utilizadas para:

1. **Documentación**: Referenciar elementos de la interfaz en la documentación
2. **Entrenamiento**: Explicar flujos de trabajo a nuevos usuarios
3. **Marketing**: Crear materiales promocionales sobre la plataforma
4. **Desarrollo**: Referencia visual para el equipo de desarrollo

## Guía de Nomenclatura

Los archivos siguen la siguiente convención de nomenclatura:

```
[módulo]_[submodulo]_[función]_v[versión].png
```

Ejemplos:
- `facturacion_creacion_formulario_v1.0.png`
- `proyectos_calendario_vista_v2.1.png`
- `automatizacion_flujo_notificaciones_v1.2.png`

## Contribuciones

Al agregar nuevas capturas de pantalla:

1. Siga la convención de nomenclatura establecida
2. Use una resolución mínima de 1280x720 píxeles
3. Oculte información sensible o de clientes reales
4. Actualice este README si agrega una nueva categoría de capturas

## Herramientas Recomendadas

Para crear capturas de pantalla de alta calidad:

- **macOS**: Comando + Shift + 4, seguido de Espacio para capturar ventanas
- **Windows**: Herramienta Recorte y anotación (Win + Shift + S)
- **Herramientas adicionales**: LightShot, Snagit o ShareX para capturas avanzadas
